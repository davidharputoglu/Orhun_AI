# Orhun AI 📱

**Assistant personnel AI intelligent, privé et multi-appareil.**

Orhun AI est un assistant Android conçu pour fonctionner prioritairement hors ligne, garantissant une confidentialité maximale tout en offrant des fonctionnalités avancées d'intelligence artificielle. Il est particulièrement optimisé pour le support des langues turciques et utilise une interface innovante sous forme d'overlay flottant.

## ✨ Fonctionnalités Principales

- **IA 100% Offline** : Utilise des modèles LLaMA 3 / Mistral quantifiés localement.
- **Support Multilingue Turcique** : Support complet de plus de 30 langues et dialectes turciques, incluant les runes d'Orkhon.
- **Interface Overlay Flottant** : Accès rapide via un robot interactif déplaçable.
- **Contrôle d'Applications** : Interaction avec YouTube, Spotify, WhatsApp, etc. (via Accessibilité).
- **Rédaction & Documents** : Génération de lettres, emails et rapports avec apprentissage de votre style d'écriture.
- **Gestion des Contacts** : Recherche phonétique intelligente (ex: "Emre Cep" ≈ "aimerais").
- **Clonage Vocal Hors Ligne** : L'IA peut parler avec votre propre voix après un court entraînement local.
- **Multi-Device** : Optimisé pour Smartphones, Tablettes (Stylet) et Android TV.

## 🛡️ Confidentialité & Éthique

- **Zéro Cloud par défaut** : Vos données, vos contacts et votre voix restent sur votre appareil.
- **Open Source** : Sous licence GPL-3.0.
- **Gratuit** : Pas de publicités, pas d'achats in-app, pas de version premium.

## 🚀 Installation (Bientôt disponible)

Orhun AI sera bientôt disponible sur GitHub Releases et F-Droid.

## 🛠️ Stack Technique

- **Langage** : Kotlin 2.0+
- **UI** : Jetpack Compose
- **IA** : GGUF (4-bit/8-bit quantifié)
- **Base de données** : Room (SQLite)
- **Permissions** : Foreground Service, WindowManager, Accessibility Service
